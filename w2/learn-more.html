HTML Tags Listed from W3 Schools:

    I want to learn how include or embed a video into a website.

  • Area

    "This tag defines an area inside an image map (an image map is an image with clickable areas). This element is always nested inside a 'map' tag."

  • Base

    "This tag specifies the base URL and/or target for all relative URLs in a document. This tag must have either an 'href' or a target attribute present, or both. There can only be one single 'base' element in a document, and it must be inside the 'head' element."

    I have not used the 'base' tag yet. It would be good to learn how it functions.
